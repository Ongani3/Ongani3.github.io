# Page Descriptions
- **Home Page:** The main landing page that introduces Onga's Tech and provides an overview of services.
- **Services Page:** A detailed description of the various services offered by Onga's Tech, including device sales, repairs, and custom PC building.
- **Contact Page:** A form for users to get in touch with Onga's Tech for inquiries or support.

# Summary of Features Used
- **Selectors:** Various CSS selectors were used to style elements, including class and ID selectors.
- **Flex/Grid:** Flexbox was used for the navigation bar and layout of the home page, while CSS Grid was used for the services section to create a responsive layout.
- **Media Queries:** Media queries were implemented to ensure the website is responsive and adapts to different screen sizes.
- **Animation:** CSS animations were used for fading in elements on the page for a smoother user experience.
